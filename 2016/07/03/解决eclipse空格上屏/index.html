解决eclipse空格上屏

发表于 2016-07-03 更新于 2019-04-20 分类于 java

下载eclipse源码

首先下载eclipse的源码，地址eclipse源码， 里面有org.eclipse.jface.text 这个插件的源码。

导入插件

在eclipse中选择 File -> import -> Plug-ins and Fragments，点击 next，然后选择下载下来的 eclipse/plugins

点击 next 然后在输入框中搜索 jface.text ,点击 Add，然后finish即可导入到eclipse中

导出

最后就是导出修改后的插件，右键点击你的 workspace 里的工程，选择 Export－> Deployable plugins and fragments，点击 Next，在 Destination 中的 Directory，选择一个要保存插件的目录，然后 Finish。就会在你所选的目录下产生一个新的 plugins 目录，里面有一个jar文件，用它替换掉你使用的 Eclipse 目录下的 plugins 里面对应的org.eclipse.jface.text_*.jar，各个版本的版本号不太一样，替换之后重启 Eclipse。
